# InsightAI

AI-powered Data Analyst Agent built using Python, Streamlit, Pandas, Plotly, and Gemini API.

## Features

* Upload CSV datasets
* Dataset Preview
* Quality Report
* Statistical Summary
* Interactive Charts
* AI Generated Insights

## Tech Stack

* Python
* Streamlit
* Pandas
* Plotly
* Gemini API

## Future Enhancements

* Excel Support
* PDF Report Generation
* Chat with Dataset
* Forecasting
* Multi-Agent Architecture

## How to Run

1. Install dependencies

pip install -r requirements.txt

2. Run the application

streamlit run backend/app.py
